3. What is a LocalStorage and SessionStorage? How do they differ?
LocalStorage or SessionStorage is a client-side storage object that allows you to st ore string values in it.
While LocalStorage's saved values remains even when the browser is closed and opened again the SessionStorage's values gets cleared after the 
browser tab is closed. LocalStorage's object is bound to same origin which means data is shared with all tabs/pages/url's that share same origin while SessionStorage's object is bound
to it's browser tab. Both storages will contain same data as before even after page refresh.

4. What is CDN? Why is it useful?
CDN - content delivery server is a group of servers spread out over many locations.
These servers store duplicate copies of data so that servers can fulfill data requests based on which servers are closest to the respective end-users
CDNs are used widely for delivering stylesheets and Javascript files (static assets) of libraries like Bootstrap, jQuery etc.
